# AI Pathfinder – Uninformed Search Visualizer

## GOOD PERFORMANCE TIME APP

This project implements and visualizes six uninformed search algorithms in a 2D grid environment using Python and Jupyter Notebook.

## Implemented Algorithms

- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Uniform Cost Search (UCS)
- Depth-Limited Search (DLS)
- Iterative Deepening DFS (IDDFS)
- Bidirectional Search

## Features

- Step-by-step animated GUI using Matplotlib
- Frontier node visualization
- Explored node visualization
- Final path highlighting
- Static obstacle handling

## How to Run

1. Install dependencies:
   pip install matplotlib numpy jupyter

2. Open the notebook
3. Run all cells
